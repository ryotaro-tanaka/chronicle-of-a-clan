## Core pillars (Design Intent)

### Clan management is the core game.
You repeatedly make decisions around hiring, equipment, party composition, and ongoing clan upkeep.

### Manual time advancement is the progression mechanism.
Advancing time runs daily processing and commits outcomes.
Over time, several parts of the game change (e.g., quest availability, hiring candidates, active quest progress, equipment crafting/upgrade progress).

> Manual time advancement is not a goal by itself—it exists to drive decision-making and resolution.

### Growth is the primary fun.
Members become stronger, equipment improves, and tougher challenges become achievable over time.

### The story emerges from systems and logs.
Instead of heavy hand-written narrative, the game records “what happened” as chronicle logs.
Quest names, equipment names, and monster names are generated by combining category-based vocab elements.

### Randomness exists, but saves are reproducible.
Equipment crafting/upgrades and monster stats include randomness, so the game is not strictly deterministic.
However, the full save state is stored in a local file, making a given saved state reproducible.

### Responsibilities are strictly separated.
Core owns rules and state; UI owns input and presentation.
The UI must not contain game rules.

## Experience at a glance (No numbers, no tuning)

A typical session repeats this loop:

1. **Check status**: roster, resources, active quests, recent logs
2. **Plan**: hire/dismiss, adjust equipment, choose quests, form parties
3. **Commit**: accept quests and start crafting/upgrading gear if needed
4. **Advance time**: run daily processing for some number of in-game days
5. **Review outcomes**: progress, injuries, rewards, and chronicle logs
6. **Adjust**: rotate members, recover, refine loadouts, and prepare the next attempts

## Goals

- Short-term: strengthen members and gear, clear tougher hunts
- Mid-term: progress through regions and increase guild rank
- Long-term (win condition): reach **#1 on the Fame ranking**
